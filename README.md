# Gen Z Mental Health Dashboard

An interactive Streamlit dashboard that analyzes student burnout, screen time, sleep patterns, and academic performance.

## Features

* KPI Cards
* Interactive Filters
* Burnout Analysis
* Correlation Heatmap
* Machine Learning Prediction
* Recommendation Engine

## Technologies Used

* Python
* Pandas
* Seaborn
* Scikit-Learn
* Streamlit

## Dataset

150,000 student records containing mental health and academic indicators.
